# SPA-Universe

## Pré-requisitos

Antes de começar, certifique-se de ter o Node.js instalado em sua máquina. Você pode baixá-lo em: [https://nodejs.org/](https://nodejs.org/)

## Instalação

rode o comando ``` npm install``` para instalar as dependências do projeto.
